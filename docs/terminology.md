# قاموس المصطلحات — QMS جمعية البر بصبيا

> مرجع موحَّد للمصطلحات المستخدمة في الواجهة والتوثيق.
> الهدف: نفس المعنى = نفس الكلمة، في كل مكان.
> آخر تحديث: 2026-04-19

---

## قاعدة عامة

- **العربية أولاً** في كل نص موجّه للمستخدم. الرموز الإنجليزية (NCR, KPI, RBAC) تبقى حيث هي جزء من إجراء رسمي.
- **لا تخلط** مترادفين لنفس المفهوم (اعتماد/إقرار/موافقة — اختر واحداً).
- **عنوان رئيسي + سطر فرعي** لكل صفحة: العنوان يوصف *ما هو*، الفرعي يشرح *ماذا يفعل المستخدم هنا*.

### وضوح التواريخ والأرقام

- **التواريخ**: ميلادي بصيغة `يوم/شهر/سنة` (مثال: `19/04/2026`). لا تخلط مع هجري في نفس الصفحة.
- **الأرقام**: `ar-SA` locale — فاصل الآلاف عربي تلقائي. الكسور العشرية رقمين على الأكثر.
- **العملة**: `1,250 ر.س` (رمز ريال سعودي عربي). لا تستخدم `SAR` في المخرجات للمستخدم.
- **النِسَب**: `87%` (بدون فراغ قبل %).
- **الوقت**: `14:30` (24 ساعة) — الأوضح للسجلات الرسمية.

### الحالة الفارغة (Empty States)

| السياق | النص المعتمد |
|-------|-------------|
| لا توجد سجلات | **لا توجد بيانات لعرضها بعد** |
| لم يتم العثور على نتائج بحث | **لا توجد نتائج مطابقة لبحثك** |
| انتظر تحميل | **⏳ جارٍ التحميل…** |
| فشل عملية | **تعذّر التنفيذ — {سبب}** |
| نجاح | **✅ تم {الفعل} بنجاح** |

## عناوين الصفحات (قالب)

```
[عنوان رئيسي]              ← ما هو (اسم الوحدة)
سطر فرعي يشرح الهدف        ← ماذا يفعل المستخدم هنا
```

أمثلة:
- **الشكاوى** — متابعة الشكاوى الواردة ومعالجتها ضمن SLA 14 يوماً (ISO 9.1.2)
- **مؤشرات الأداء** — إدخال ومتابعة القراءات الشهرية حسب محاور BSC
- **حالات عدم المطابقة** — تسجيل الانحرافات، تحليل الأسباب، تنفيذ الإجراءات التصحيحية (ISO 10.2)

## استخدام هذا القاموس

1. **عند إضافة صفحة جديدة**: راجع هذا القاموس أولاً.
2. **عند التعديل**: إن وجدت مصطلحاً مختلفاً — عدّله للمعتمد هنا.
3. **عند الاختلاف**: افتح PR لتحديث هذا القاموس *أولاً*، ثم طبّق التغيير.
4. **قبل الإطلاق**: مراجعة لغوية سريعة بواسطة مدير الجودة على كل نص ظاهر للمستخدم.
