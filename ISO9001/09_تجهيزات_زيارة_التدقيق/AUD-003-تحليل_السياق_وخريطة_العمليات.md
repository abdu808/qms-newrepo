- المشتريات والموردين
- المالية والمحاسبة
- التقنية والأنظمة

# الجزء الخامس: كيف يسير العمل عندنا (خريطة العمليات)

## 6. ثلاث فئات من الأعمال

### أ. أعمال القيادة
> ما يقوم به مجلس الإدارة والمدير التنفيذي ومدير الجودة

- التخطيط الاستراتيجي
- اجتماع مراجعة الإدارة
- إدارة المخاطر
- التحسين المستمر

### ب. الأعمال الأساسية (التي نوجد لأجلها)

**خط خدمة المستفيدين:**
```
استقبال الطلب  →  دراسة الحالة  →  تقديم الخدمة  →  متابعة الأثر
```

**خط التبرعات:**
```
استقطاب التبرع  →  استلام التبرع  →  تسجيله  →  صرفه في مكانه
```

### ج. الأعمال المساندة

| الإدارة المساندة | عملها |
|---|---|
| الموارد البشرية | توظيف، تدريب، رواتب |
| المتطوعون | تجنيد، تدريب، تنظيم |
| المشتريات | اختيار موردين، شراء |
| المالية | محاسبة، تقارير |
| التقنية | حماية الأنظمة، دعم فني |
| ضبط الوثائق | حفظ السجلات |

## 7. ملخّص لكل عملية رئيسية

| العملية | المدخلات | المخرجات | كيف نقيس نجاحها |
|---|---|---|---|
| استقبال طلب مستفيد | طلب + مستندات | ملف مفتوح | عدد الطلبات، سرعة المعالجة |
| دراسة الحالة | الملف + المعايير | قرار قبول/رفض | عدد القرارات في الموعد |
| تقديم الخدمة | قرار قبول | خدمة فعلية | رضا المستفيد |
| متابعة الأثر | بيانات الخدمة | تقرير الأثر | تحسّن وضع المستفيد |
| استقطاب التبرعات | حملات | تبرعات | حجم التبرعات |
| استلام التبرع | تبرع وارد | إيصال + قيد | دقة وسرعة التسجيل |
| صرف التبرع | موازنة معتمدة | صرف موثّق | الالتزام بالموازنة |

---

## 8. منهج عمل بسيط لكل عملية (دورة التحسين)

كل عملية نتّبع فيها أربع خطوات:
1. **خطّط** — حدّد الهدف والمسؤول
2. **نفّذ** — قم بالعمل حسب الإجراء
3. **راجع** — قس النتيجة
4. **حسّن** — عدّل ما يحتاج تعديل

## 9. أبرز المخاطر وكيف نواجهها

| المخاطرة | كيف نواجهها |
|---|---|
| تسرّب بيانات المستفيدين | سياسة حماية المستفيدين + صلاحيات وصول |
| تسجيل تبرع غير دقيق | تدقيق مزدوج + نظام إلكتروني |
| صرف خارج الموازنة | اعتمادات متعددة قبل الصرف |
| توظيف غير مناسب | معايير اختيار + فترة تجربة |
| اختراق الأنظمة | نسخ احتياطية + جدران حماية |
| نقص في التبرعات | تنويع مصادر التمويل |
