# 13 — قائمة المراجعة السنوية
**المالك:** المدير العام + مدير الجودة + مجلس الإدارة
**الإيقاع:** يناير من كل سنة (لكامل السنة السابقة).
**الوقت المتوقع:** أسبوع كامل من العمل الممنهج.

## 📆 خطة الأسبوع

### اليوم 1: سحب البيانات وتجهيز الحزمة (الجودة)
- [ ] تصدير كل KPIs لكامل السنة (12 نقطة قياس لكل مؤشر).
- [ ] تصدير كل الـ NCR للسنة (فتح/إغلاق، زمن دورة، تصنيف).
- [ ] تصدير سجل المخاطر (snapshot بداية العام + نهاية العام).
- [ ] تصدير كل الشكاوى ونتائج الرضا.
- [ ] تصدير كل التدقيقات الداخلية للسنة + نتائج.
- [ ] تصدير بيانات المستفيدين (الداخلون الجدد + الخارجون).
- [ ] تصدير تقرير التبرعات السنوي.
- [ ] إنشاء مجلد **Annual Review 2026** في شاشة Documents.

### اليوم 2: تحليل الأداء (الجودة + كل الأقسام)
- [ ] تحليل تحقيق الأهداف السنوية (Objectives):
  - ما نسبة الأهداف المُحقَّقة؟
  - ما نسبة المتحققة جزئياً؟
  - ما الأهداف الفاشلة؟ لماذا؟
- [ ] تحليل الاتجاهات (Trends):
  - هل المؤشرات تتحسن، تثبت، أم تتراجع؟
  - مقارنة مع السنة السابقة لكل KPI.
- [ ] تحليل الانحرافات:
  - أي مؤشر انحرف عن الهدف بـ ≥15%؟ ما الأسباب؟

### اليوم 3: تقييم الموردين والشركاء
- [ ] تقييم سنوي لكل مورّد معتمد (Supplier Eval).
- [ ] قرار لكل مورد: تجديد، مشروط، إنهاء.
- [ ] تحديث قائمة الموردين المعتمدين لعام 2027.
- [ ] تقييم الشركاء الإستراتيجيين (الجهات الحكومية، الرعاة).

### اليوم 4: تقييم الموظفين والكفاءة
- [ ] تقييم أداء كل موظف (HR).
- [ ] تحديث متطلبات الكفاءة بناءً على الواقع.
- [ ] خطة التطوير الفردية لكل موظف.
- [ ] مراجعة الوصف الوظيفي لكل منصب.

### اليوم 5: المراجعة الإدارية الرسمية (P-13)
- [ ] اجتماع المدير العام + مديرو الأقسام + لجنة الجودة (4-6 ساعات).
- [ ] استعراض جدول المراجعة الإدارية (مدخلات إلزامية):
  1. حالة الإجراءات من المراجعة السابقة
  2. التغييرات في السياق الداخلي/الخارجي
  3. تحقيق الأهداف
  4. أداء العمليات ومطابقة المنتجات/الخدمات
  5. عدم المطابقات والإجراءات التصحيحية
  6. نتائج المراقبة والقياس (KPIs)
  7. نتائج التدقيق
  8. أداء الموردين الخارجيين
  9. كفاية الموارد
  10. فعالية الإجراءات المتخذة لمعالجة المخاطر والفرص
  11. فرص التحسين
  12. رضا المستفيدين والمتبرعين
- [ ] توثيق كل قرار ومسؤول في شاشة **Management Review**.

### اليوم 6-7: التخطيط للعام الجديد
- [ ] صياغة الأهداف السنوية 2027 (Objectives).
- [ ] تحديث الخطة الإستراتيجية.
- [ ] مراجعة سياسة الجودة — هل تحتاج تحديث؟
- [ ] تحديث سجل المخاطر بالمخاطر الجديدة للعام.
- [ ] إعداد خطة التدقيق الداخلي للسنة القادمة.
- [ ] إعداد خطة التدريب السنوية.
- [ ] إعداد موازنة الجودة (الوقت، المال، الموارد).

## ✅ مخرجات سنوية إلزامية

| المخرج | الوثيقة في النظام | المدة المرجعية |
|--------|--------------------|----------------|
| تقرير أداء سنوي | Documents · REPORT | ≥ 30 صفحة |
| محضر المراجعة الإدارية السنوية | Management Review | موقّع من المدير العام |
| خطة الأهداف 2027 | Objectives | معتمدة |
| خطة التدقيق 2027 | Audits | معتمدة |
| خطة التدريب 2027 | Training | معتمدة |
| قائمة الموردين المعتمدين 2027 | Suppliers | مُحدَّثة |
| سياسة الجودة (مُعتمَدة أو مُحدَّثة) | Quality Policy | معتمدة |
| سجل المخاطر 2027 | Risks | مُحدَّث |
| تقرير رضا المستفيدين السنوي | Surveys · REPORT | علني |
| التقرير المالي السنوي | (خارج QMS) | مُدقَّق |

## 🎯 معايير نجاح السنة
- ≥ 85% من الأهداف مُحقَّقة.
- ≥ 85% معدل رضا المستفيدين.
- 100% من NCR مغلقة بفعالية قبل نهاية السنة أو مُجدوَلة.
- 100% تنفيذ خطة التدقيق الداخلي.
- 0 حالة تعليق عضوية ISO أو انسحاب من الشهادة.
- 0 مخالفة جسيمة من الجهات الرقابية.

## 🗂️ الأرشفة
- جميع الوثائق السنوية تُؤرشف في مجلد `Annual Review [السنة]` داخل شاشة Documents.
- لا يُسمح بتعديلها بعد الأرشفة — فقط قراءة (immutable).
- تُحتفظ لمدة ≥ 7 سنوات (متطلب قانوني ومحاسبي).

## 🔄 ربط دوري مع ISO Certification
- قبل 90 يوم من زيارة جهة الاعتماد: تدقيق داخلي شامل (Full scope internal audit).
- قبل 30 يوم: مراجعة إدارية مُخصَّصة للتحضير.
- أثناء الزيارة: كل الوثائق جاهزة في شاشة "تجهيزات زيارة التدقيق" ضمن النظام.
