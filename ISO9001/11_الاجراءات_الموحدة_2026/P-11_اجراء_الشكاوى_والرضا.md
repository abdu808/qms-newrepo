# P-11 — إجراء الشكاوى وقياس الرضا
**المالك:** خدمة العملاء + مدير الجودة · **البند:** 9.1.2

## 1. الغرض
استقبال ومعالجة الشكاوى بشفافية وفعالية، وقياس رضا المستفيدين والمتبرعين باستمرار.

## 2. مصادر الشكاوى
| المصدر | الوسيلة |
|-------|---------|
| مستفيد | مقابلة · اتصال · صندوق شكاوى · موقع · واتساب |
| متبرع | اتصال · بريد |
| موظف | خط ساخن داخلي |
| جهة شريكة | رسمي |
| مجتمع | سوشيال ميديا · صحافة |

## 3. الخطوات

### 3.1 الاستقبال
- موظف الاستقبال يُنشئ **Complaint** فوراً:
  - اسم المشتكي + بيانات التواصل
  - موضوع الشكوى
  - تفاصيلها
  - المستوى (خطورة):
    - منخفضة: انزعاج بسيط
    - متوسطة: عدم رضا يؤثر في الخدمة
    - مرتفعة: تظلّم أو إخلال بحق
- إشعار فوري لمدير الجودة + مدير القسم المعني.

### 3.2 التصنيف
- نوع الشكوى: خدمة / سلوك / تأخير / مالي / خصوصية / أخرى.
- ربطها بالقسم المسؤول (assigneeId).

### 3.3 المعالجة
| الخطورة | هدف الاستجابة | هدف الحل |
|---------|----------------|----------|
| منخفضة | 24 ساعة | 7 أيام |
| متوسطة | 8 ساعات | 3 أيام |
| مرتفعة | 2 ساعة | 24 ساعة |

- تحليل السبب الجذري (root cause).
- حل للمشتكي (resolution).
- تواصل مع المشتكي قبل الإغلاق للتأكد من رضاه.

### 3.4 الإغلاق
- بعد الحل + تأكيد رضا المشتكي → status=RESOLVED.
- بعد أسبوعين بلا اعتراض → CLOSED.
- إذا كانت الشكوى تشير إلى خلل نظامي → فتح NCR وربط.

### 3.5 التحليل الدوري
- شهرياً: تحليل الشكاوى (تكرار، فئات، أقسام).
- ربع سنوي: تقرير اتجاهات.
- أي نمط متكرر → فتح NCR + إجراء تصحيحي نظامي.

## 4. قياس الرضا
### 4.1 استبيان رضا المستفيدين
- بعد كل خدمة رئيسية: استبيان قصير (3-5 أسئلة).
- ربع سنوي: استبيان شامل.

### 4.2 استبيان رضا المتبرعين
- نصف سنوي لكل المتبرعين.

### 4.3 تحليل النتائج
- تقرير مفصَّل بالتقسيم:
  - حسب القسم
  - حسب نوع الخدمة
  - حسب فئة المستفيد
- أي درجة < 70% تستدعي تحقيقاً.

## 5. قواعد
- **لا شكوى تُرفض بدون دراسة** حتى التافهة.
- **سرّية هوية المشتكي** إن طلب ذلك.
- **لا ضغط على المشتكي** للسحب.
- **الردود بلغة المشتكي** لا مصطلحات فنية.

## 6. السجلات
- Complaints (كل الشكاوى)
- Surveys + نتائج
- محاضر التحليل الشهري
- مراسلات مع المشتكين

## 7. KPIs
- Q-04: متوسط زمن المعالجة ≤ **48 ساعة**
- Q-06: رضا المستفيدين ≥ **85%**
- نسبة الشكاوى المُغلَقة = **100%**
- عدد الشكاوى المتكررة من نفس النوع = **يقل سنوياً**
