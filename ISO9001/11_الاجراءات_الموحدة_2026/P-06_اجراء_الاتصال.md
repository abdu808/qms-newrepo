# P-06 — إجراء الاتصال الداخلي والخارجي
**المالك:** مدير الجودة + العلاقات العامة · **البند:** 7.4

## 1. الغرض
تنظيم الاتصال بحيث يصل للشخص المناسب في الوقت المناسب بالشكل المناسب.

## 2. أنواع الاتصال
| النوع | الجمهور | الوسيلة | التكرار |
|------|---------|--------|---------|
| إداري داخلي | الموظفون | رسالة بريد + لوحة | حسب الحاجة |
| توعوي | المجتمع | موقع + سوشيال ميديا | أسبوعي |
| تشغيلي للأقسام | مديرو الأقسام | اجتماع + نظام | أسبوعي |
| مع المتبرعين | المتبرعون | SMS + بريد | بعد كل تبرع + شهري |
| مع المستفيدين | المستفيدون | اتصال + SMS | حسب الحالة |
| مع جهات رقابية | الجهات الحكومية | رسمي مختوم | حسب الطلب |
| مع الإعلام | الصحافة | بيانات صحفية | حسب الحدث |

## 3. الخطوات
1. **إعداد خطة اتصال سنوية** في شاشة **Communication Plan**:
   - ما نتواصل بشأنه
   - مع من (جمهور)
   - متى (تردد)
   - كيف (قناة)
   - من مسؤول
2. **تنفيذ** الخطة — كل نشاط اتصال يُسجَّل.
3. **متابعة** ربع سنوية — نسبة الإنجاز.
4. **قياس الفعالية**:
   - داخلي: استبيان موظفين عن وضوح الرسائل
   - خارجي: معدل الانفتاح (open rate) · الاستجابة · الذكر الإيجابي

## 4. قواعد
- الاتصال الرسمي باسم الجمعية يمر عبر العلاقات العامة.
- المعلومات الحساسة (بيانات مستفيدين، مالية) لا تُنشَر إطلاقاً.
- الأزمات: بيان موحّد من المدير العام فقط.

## 5. السجلات
- Communication Plan (كل الأنشطة)
- سجل الرسائل الصادرة
- البيانات الصحفية
- محاضر الاجتماعات الداخلية

## 6. KPIs
- C-01: تنفيذ خطة الاتصال ≥ **90%**
- C-02: نسبة إقرار الوثائق الجديدة = **100% خلال 7 أيام**

## 7. الربط مع النظام
- شاشة `/communication` · POST /api/communication
- شاشة `/documents` للاتصال الرسمي
- شاشة `/surveys` لاستطلاعات الرأي

## 8. المراجع
