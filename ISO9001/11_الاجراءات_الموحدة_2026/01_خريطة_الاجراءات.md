# 01 — خريطة الإجراءات الشاملة
**البند ISO 9001 المرجعي:** 4.4 (نظام إدارة الجودة وعملياته)

## 1. تصنيف العمليات

| النوع | التعريف | أمثلة من الجمعية |
|------|---------|------------------|
| **عمليات إستراتيجية** | توجّه المنظمة | التخطيط الإستراتيجي، المراجعة الإدارية، سياسة الجودة |
| **عمليات تشغيلية** | تنتج القيمة للمستفيد/المتبرع | استقبال التبرعات، تسجيل المستفيدين، صرف المساعدات، تقديم الخدمات |
| **عمليات دعم** | تمكّن العمليات التشغيلية | الموارد البشرية، المشتريات، المالية، تقنية المعلومات، ضبط الوثائق |
| **عمليات قياس وتحسين** | تراقب وتحسّن | التدقيق الداخلي، الشكاوى، عدم المطابقة، رضا المستفيدين |

## 2. الخريطة العملياتية (تدفّق القيمة)

```
┌──────────────────── مُدخَلات ────────────────────┐
│  المتبرعون · المستفيدون · الموظفون · الموردون    │
└──────────────────────────────────────────────────┘
            │
            ▼
┌──────────────── عمليات استراتيجية ───────────────┐
│  P-01 السياق  ·  P-02 السياسة والأهداف         │
│  P-13 المراجعة الإدارية                         │
└──────────────────────────────────────────────────┘
            │
            ▼
┌──────────────── عمليات تشغيلية ──────────────────┐
│  P-07 التخطيط التشغيلي                          │
│  P-09 التبرعات → P-10 المشتريات                 │
│                    ↓                            │
│                 P-08 خدمة المستفيدين            │
└──────────────────────────────────────────────────┘
            │
            ▼
┌──────────────── عمليات دعم ──────────────────────┐
│  P-04 الوثائق  ·  P-05 الموارد البشرية          │
│  P-06 الاتصال                                   │
└──────────────────────────────────────────────────┘
            │
            ▼
┌──────────── قياس · تحسين · مخاطر ────────────────┐
│  P-03 المخاطر · P-11 الشكاوى · P-12 التدقيق     │
│  P-14 عدم المطابقة · P-15 التحسين المستمر        │
└──────────────────────────────────────────────────┘
            │
            ▼
┌──────────────────── مُخرَجات ────────────────────┐
│  مستفيد مخدوم · متبرع راضٍ · تقرير إداري موثّق  │
└──────────────────────────────────────────────────┘
```

## 3. مصفوفة العملية × القسم المالك

| الإجراء | القسم المالك الرئيسي | أقسام مشاركة | شاشة النظام |
|---------|----------------------|---------------|--------------|
| P-01 السياق | الجودة | التخطيط الإستراتيجي، الإدارة التنفيذية | SWOT · Interested Parties · Strategic Goals |
| P-02 السياسة والأهداف | الإدارة التنفيذية | الجودة | Quality Policy · Objectives · Strategic Goals |
| P-03 المخاطر | الجودة | كل الأقسام | Risks |
| P-04 الوثائق | الجودة | كل الأقسام | Documents |
| P-05 الموارد البشرية | الموارد البشرية | الجودة، الأقسام | Training · Competence · Users |
| P-06 الاتصال | الجودة / العلاقات العامة | كل الأقسام | Communication Plan |
| P-07 التخطيط التشغيلي | كل قسم | الجودة | Operational Activities · KPI Entries |
| P-08 خدمة المستفيدين | الخدمات الاجتماعية | المالية، الخدمات | Beneficiaries · Programs · Surveys |
| P-09 التبرعات | التبرعات | المالية | Donations · Donation Evals |
| P-10 المشتريات | المشتريات | المالية، الجودة | Suppliers · Supplier Evals |
| P-11 الشكاوى | خدمة العملاء | الجودة، القسم المعني | Complaints · Surveys |
| P-12 التدقيق | الجودة | كل الأقسام | Audits |
| P-13 المراجعة الإدارية | المدير العام | الجودة، كل الأقسام | Management Review |
| P-14 عدم المطابقة | الجودة | القسم المعني | NCR |
| P-15 التحسين المستمر | الجودة | كل الأقسام | Objectives · NCR · Risks |

## 6. معايير التقاطع مع النظام الإلكتروني
كل عملية في هذه الخريطة **يجب** أن تقابلها شاشة وظيفية في QMS. حالات عدم التطابق تُسجَّل كـ **NCR** من نوع "Process-System Gap" وتُعالج وفق P-14.
