# 11 — قائمة المراجعة الأسبوعية
**المالك:** مدير الجودة + مديرو الأقسام
**اليوم المقترح:** كل يوم أحد (بداية الأسبوع) - 45 دقيقة.

## 👤 مدير الجودة (يوم الأحد صباحاً)

### 📊 مؤشرات أساسية (Dashboard)
- [ ] Q-01: عدد NCR المفتوحة > 30 يوم = **هدف: 0**
- [ ] Q-04: متوسط زمن معالجة الشكوى هذا الأسبوع = **هدف: ≤ 48 ساعة**
- [ ] Q-09: عدد الوثائق المنتهية صلاحيتها = **هدف: 0**
- [ ] عدد السجلات في حالة SUBMITTED > 72 ساعة (أي تأخير مراجعة) = **هدف: 0**

### 🔍 فحوصات دورية
- [ ] مراجعة كل الـ NCR المفتوحة (NCR · status≠CLOSED) وحالاتها:
  - هل السبب الجذري موثّق؟
  - هل تاريخ الاستحقاق مناسب؟
  - أي منها متأخر؟ اتصال بالمسؤول.
- [ ] مراجعة المخاطر الحرجة (Risks · level="حرج") — هل لها خطة معالجة؟
- [ ] مراجعة خطة التدقيق الداخلي — أي تدقيق هذا الأسبوع؟
- [ ] فحص سجل التدقيق (Audit Log) — بحث عن أنشطة غير طبيعية (حذف متكرر، تحديث بعد ساعات الدوام).
- [ ] مراجعة الشكاوى المفتوحة — أي شكوى > 7 أيام بدون رد؟

### 📝 توثيق وتواصل
- [ ] إرسال تقرير أسبوعي موجز للمدير العام (عبر قسم الاتصالات في النظام).
- [ ] تحديث شاشة "الوثائق" إن طرأ أي تعديل.
- [ ] متابعة خطة الاتصال (Communication Plan) — هل التواصل الأسبوعي المخطط تم؟

## 👤 مديرو الأقسام (كل يوم أحد)

### المالية
- [ ] مطابقة إجمالي التبرعات في النظام (Donations sum) مع كشف البنك الأسبوعي.
- [ ] التحقق أن 100% من التبرعات في حالة VERIFIED خلال 48 ساعة من الاستلام.
- [ ] مراجعة أي تبرع > 10,000 ريال — هل اعتمده المدير العام؟

### الموارد البشرية
- [ ] مراجعة الموظفين الذين لم يسجّلوا دخولاً في النظام هذا الأسبوع.
- [ ] فحص متطلبات الكفاءة (Competence Requirements) — أي موظف تنتهي شهادته قريباً؟
- [ ] تحديث سجلات التدريب (Training Records) الأسبوعية.

### التبرعات والموارد
- [ ] مقارنة تبرعات هذا الأسبوع مع نفس الأسبوع من العام الماضي.
- [ ] متابعة المتبرعين الدائمين (Recurring Donors) — أي منهم توقف؟
- [ ] تحديث قائمة الحملات (Programs) النشطة.

### الخدمات الاجتماعية
- [ ] عدد الطلبات الجديدة للمستفيدين هذا الأسبوع.
- [ ] عدد المستفيدين الذين صُرفت لهم مساعدات.
- [ ] متوسط زمن البت في طلب المستفيد (هدف ≤ 7 أيام).
- [ ] مراجعة حالات "تحت الدراسة" المُعلَّقة > 7 أيام.

### المشتريات
- [ ] مراجعة طلبات الشراء المعلَّقة هذا الأسبوع.
- [ ] فحص الموردين الذين يحتاجون تقييم (تقييمهم السنوي خلال 30 يوم).
- [ ] مراجعة عدد العروض لكل طلب شراء (هدف ≥ 3).

### تقنية المعلومات
- [ ] التحقق من النسخ الاحتياطية اليومية (backups).
- [ ] فحص سجلات الدخول للنظام — أي دخول مشبوه؟
- [ ] مراجعة الصلاحيات — أي موظف غادر لكن لا يزال حسابه نشطاً؟
- [ ] تحديث نظام التشغيل والتطبيقات.

### العلاقات العامة
- [ ] مراجعة الخطة الإعلامية الأسبوعية.
- [ ] تحديث الموقع الإلكتروني ووسائل التواصل.
- [ ] الرد على استفسارات الصحافة/المجتمع.

### الجودة (المالك الأسبوعي)
- [ ] تصدير تقرير الأنشطة التشغيلية (Operational Activities) — نسبة الإنجاز.
- [ ] اجتماع لجنة الجودة القصير (Weekly Stand-up 30 دقيقة).

## 🔔 نقاط فحص حرجة أسبوعية

| النقطة | المعيار | إجراء إذا لم يُستوفَ |
|--------|---------|----------------------|
| تبرعات غير مُعتمَدة | 0 | الاتصال بمدير التبرعات فوراً |
| NCR بدون سبب جذري | 0 | تذكير المسؤول بإضافة السبب |
| وثائق منتهية الصلاحية | 0 | سحب الوثيقة ومراجعتها |
| مخاطر حرجة بدون معالجة | 0 | تصعيد للمدير العام |
| شكاوى بلا رد > 48 ساعة | 0 | جلسة طارئة مع مدير القسم |
| موظف لم يُدخِل بيانات > 3 أيام | 0 | استفسار + تدريب |

## 📑 المحاضر المطلوبة
- محضر اجتماع لجنة الجودة الأسبوعي (في شاشة Documents · فئة MEETING).
- تقرير ملخص من مدير الجودة للمدير العام (في شاشة Communication Plan).
