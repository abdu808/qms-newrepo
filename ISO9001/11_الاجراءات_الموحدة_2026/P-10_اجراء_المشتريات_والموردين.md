# P-10 — إجراء المشتريات والموردين
**المالك:** قسم المشتريات · **البند:** 8.4

## 1. الغرض
ضمان أن كل ما يُشترى أو يُستعان به من خارج الجمعية يتوافق مع المتطلبات ولا يؤثر سلباً على الخدمات.

## 2. تصنيف الموردين
| النوع | أمثلة |
|------|-------|
| GOODS | مستلزمات مكتبية، أغذية، ملابس |
| SERVICES | تنظيف، صيانة، خدمات طبية |
| CONSTRUCTION | مقاولات، ترميم |
| IT_SERVICES | خدمات تقنية، استضافة |
| IN_KIND_DONOR | موردو التبرعات العينية |
| TRANSPORT | نقل، مواصلات |
| CONSULTING | استشارات |

## 3. الخطوات

### 3.1 اعتماد المورد (Qualification)
قبل التعامل مع مورد لأول مرة:
- **مستندات إلزامية**:
  - سجل تجاري ساري
  - شهادة ضريبة القيمة المضافة
  - شهادة ضمانات اجتماعية
  - شهادة زكاة ودخل
- **تقييم أولي**:
  - الجودة (منتج/عينة)
  - الأسعار (مقارنة مع السوق)
  - السمعة (مراجع)
- تُسجَّل المعلومات في **Suppliers** بحالة PENDING.
- مدير المشتريات يعتمد → APPROVED.

### 3.2 طلب الشراء
- **< 1,000 ريال**: طلب شفهي + إيصال.
- **1,000 - 10,000 ريال**: عرضان مكتوبان + اعتماد مدير القسم الطالب.
- **10,000 - 50,000 ريال**: ثلاثة عروض + اعتماد مدير المشتريات + المالية.
- **> 50,000 ريال**: طرح مناقصة + اعتماد المدير العام + مجلس الإدارة للمبالغ الكبيرة.

### 3.3 اختيار المورد
- مقارنة الأسعار + المواصفات + مدة التوريد.
- وثيقة تبرير الاختيار.
- لا يُختار المورد الأقل سعراً فقط — يُعتبر الجودة والسمعة.

### 3.4 التعاقد
- عقد مكتوب يذكر:
  - الكمية والمواصفات
  - السعر الإجمالي
  - مدة التوريد
  - شروط السداد
  - عقوبات التأخير
  - ضمان/كفالة حسنة التنفيذ (للعقود الكبيرة)

### 3.5 الاستلام والفحص
- فحص الكمية والجودة عند الاستلام.
- تسجيل Goods Receipt.
- أي عدم مطابقة → NCR فوري + إبلاغ المورد.

### 3.6 التقييم الدوري
- **سنوياً** لكل مورد معتمد:
  - الجودة (30%)
  - الالتزام بالمواعيد (25%)
  - التواصل (20%)
  - الأسعار (15%)
  - الامتثال (10%)
- درجة التقييم في **Supplier Evals** تحدد:
  - ≥ 80% → معتمد
  - 60-79% → مشروط (متابعة لصيقة)
  - < 60% → مرفوض (إنهاء التعامل)

## 4. قواعد صارمة
- **فصل المهام**: الطالب ≠ المعتمد ≠ المستلم.
- **لا مورد بقريب/صهر لموظف** يتخذ قرار الشراء (تضارب مصالح).
- تضارب المصالح يُعلَن كتابياً.
- **لا دفع مقدم إلا بضمان** (خطاب ضمان بنكي).

## 5. السجلات
- Suppliers (كل الموردين)
- Supplier Evals (كل التقييمات)
- العقود والطلبات
- إيصالات الاستلام
- تقييمات الجودة بعد كل توريد

## 6. KPIs
- P-01: الموردون المعتمدون ≥ **25**
- P-02: تقييم سنوي للموردين = **100%**
- P-03: متوسط تقييم الموردين ≥ **80%**
- P-04: طلبات بـ ≥ 3 عروض = **≥ 95%**
